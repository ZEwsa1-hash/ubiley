# «Карине 35» — сайт-приглашение

Адаптивное одностраничное приглашение в стилистике восточной сказки. Сайт собран на React, TypeScript и Vite, содержит обратный отсчёт, программу, контакты и анкету гостя с записью ответов в Google Sheets.

## Локальный запуск

Требуется Node.js 20 или новее.

```bash
npm install
copy .env.example .env.local
npm run dev
```

Проверки перед публикацией:

```bash
npm run lint
npm run build
npm run preview
```

Vite настроен с `base: /ubiley/`, поэтому локальный адрес обычно будет `http://localhost:5173/ubiley/`.

## Подключение Google Sheets

1. Создайте новую таблицу Google Sheets.
2. Откройте **Расширения → Apps Script**.
3. Скопируйте содержимое `google-apps-script/Code.gs` в редактор. При необходимости перенесите настройки из `appsscript.json`.
4. Выберите **Развернуть → Новое развертывание → Веб-приложение**.
5. Выполнение — от своего имени, доступ — для всех пользователей.
6. Скопируйте URL вида `https://script.google.com/macros/s/.../exec` в `.env.local`:

```env
VITE_RSVP_ENDPOINT=https://script.google.com/macros/s/ВАШ_ID/exec
```

После первого ответа скрипт создаст лист `Ответы` и колонки: дата и время, ФИО, присутствие, алкоголь, свой вариант, дети, количество детей. Проверить endpoint можно, открыв его URL в браузере: он вернёт JSON с `ok: true`.

> После изменения Apps Script создайте новую версию развертывания. Не добавляйте `.env.local` в Git.

## Музыка

Аудиофайл не хранится в репозитории. Перед публикацией добавьте лицензированный файл:

```text
public/music/arabskaya-noch.mp3
```

Интерфейс подготовлен для композиции «Алексей Чумаков — Арабская ночь». Воспроизведение начинается только после нажатия гостем на музыкальную кнопку. Альтернативный URL можно передать через `VITE_MUSIC_URL`.

## Публикация на GitHub Pages

Workflow `.github/workflows/deploy.yml` запускает lint, production build и публикует папку `dist` при каждом push в `main`.

1. В репозитории откройте **Settings → Pages** и выберите источник **GitHub Actions**.
2. В **Settings → Secrets and variables → Actions** создайте secret `VITE_RSVP_ENDPOINT` с URL веб-приложения Apps Script.
3. Отправьте изменения в ветку `main`.

Опубликованный адрес: `https://zewsa1-hash.github.io/ubiley/`.

## Где менять данные

Основные данные собраны в `src/eventConfig.ts`: имя, возраст, дата, часовой пояс, программа, площадка, карта, дресс-код и контакты. Изображения находятся в `public/images` и уже оптимизированы в WebP.
