# Мобильное приглашение «Карине 35»

Сайт сделан только в мобильном формате. На большом экране мобильная страница остаётся по центру и не превращается в отдельную ПК-версию.

## Запуск

Откройте `index.html` или разместите папку на любом статическом хостинге: GitHub Pages, Vercel, Netlify либо обычном хостинге.

## Google Таблица

1. Создайте Google Таблицу.
2. Откройте **Расширения → Apps Script**.
3. Вставьте содержимое файла `google-apps-script.gs`.
4. Нажмите **Развернуть → Новое развертывание → Веб-приложение**.
5. Выполнять от имени владельца. Доступ — всем, у кого есть ссылка.
6. Скопируйте URL веб-приложения.
7. Вставьте URL вида `https://script.google.com/macros/s/.../exec` в `js/config.js` в поле `googleScriptUrl`.
8. Откройте URL веб-приложения в браузере. Должно появиться сообщение `Сервис приёма ответов работает.`
9. Отправьте тестовый ответ через сайт и проверьте лист **Ответы гостей**.

Пока URL не указан или имеет неверный формат, ответы сохраняются только в `localStorage` браузера для тестирования. Сайт прямо сообщает, что Google Таблица ещё не подключена, и не показывает ложное подтверждение отправки.

После изменения кода Apps Script откройте **Развернуть → Управление развертываниями**, создайте новую версию и обновите действующее развертывание.

## Музыка

Из-за авторских прав аудиофайл не включён. Поместите легально полученный файл песни в:

`assets/audio/aleksejj-chumakov-arabskaja-noch.mp3`

Кнопка с нотой на первом экране начнёт включать и выключать музыку.

## Настройки

Дата, ссылка на Яндекс Карты и URL Google Apps Script находятся в `js/config.js`.
